# Curso de Programação Orientada a Objetos (POO)

Um projeto web educacional criado para ensinar os conceitos fundamentais da Programação Orientada a Objetos de forma prática, divertida e objetiva.

## 💻 Sobre o Projeto

Este projeto é um site de curso gratuito focado em ensinar POO com exemplos práticos em três linguagens diferentes: **Java, Python e C++**. O curso é projetado para que estudantes possam aprender no seu próprio ritmo, através de aulas diretas ao ponto, materiais complementares e exercícios práticos.

O curso aborda os quatro pilares fundamentais da Programação Orientada a Objetos:
- Encapsulamento
- Herança
- Polimorfismo
- Abstração

## 🚀 Funcionalidades e Estrutura

O curso está dividido em 4 módulos abrangendo um total de 19 aulas. O site possui:
- **Página Inicial (`index.html`)**: Apresentação do curso, estatísticas e visão geral.
- **Aulas (`aulas.html`, `aula.html`)**: Onde as aulas em vídeo ou texto são disponibilizadas de forma estruturada.
- **Exercícios (`exercicio.html`)**: Ambiente para fixação do aprendizado prático.
- **Material Complementar (`material.html`)**: Acesso a recursos adicionais (presentes na pasta `assets/Materias Complementares/`).

Além disso, o site conta com recursos modernos como alternância entre temas (Claro/Escuro) e um design responsivo.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**: Estilização do site com paleta de cores moderna, responsividade e suporte a temas.
- **JavaScript (Vanilla)**: Lógica do site, controle de temas e navegação (encontrados na pasta `js/`).
- **Phosphor Icons**: Biblioteca de ícones (carregada via CDN).
- **Google Fonts**: Utilização da fonte 'Raleway'.

## 📁 Estrutura de Diretórios

```text
├── assets/                  # Imagens e logotipos
│   └── Materias Complementares/ # Materiais de leitura e estudo de cada módulo
├── css/                     # Estilos (ex: style.css)
├── js/                      # Scripts para as páginas (ex: main.js, aula.js, exercicio.js)
├── index.html               # Página inicial do projeto
├── aulas.html               # Lista de módulos e aulas
├── aula.html                # Template/Visualização de uma aula específica
├── exercicio.html           # Página de exercícios de fixação
└── material.html            # Página listando os materiais complementares
```

## ⚙️ Como Executar

Por ser um projeto web estático, não há necessidade de configurações complexas.
1. Clone ou baixe o repositório.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. Para uma experiência completa, você também pode usar extensões como o **Live Server** (do VS Code) para emular um servidor local.

## 📚 Aprendizado

O foco principal do curso é garantir o entendimento sólido em:
1. Sintaxe e estrutura básica de classes e objetos.
2. O funcionamento da memória e ciclo de vida dos objetos nas 3 linguagens base.
3. Design Patterns e Boas Práticas (SOLID).
4. Aplicação real das regras e conceitos de POO em projetos.
